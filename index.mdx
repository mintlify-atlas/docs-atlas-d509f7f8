---
title: "Shaiya Core: C++ patching library for private servers"
description: "Shaiya Core is a modular C++ patching framework that modernizes Shaiya private servers with configurable hooks, EP6.4 support, and community-driven development."
---

Shaiya Core is a C++ patching library and hook framework for Shaiya private server research, production, and preservation. It targets the four main Shaiya binaries — `ps_game.exe`, `ps_dbAgent.exe`, `ps_login.exe`, and `Game.exe` — and provides a stable, configurable, and extensible foundation for running an Episode 6.4-compatible private server.

The project is the official successor to Shaiya Essentials. It is free, open-source under the BSD 3-Clause License, and built with community contributions. Features are controlled through `.ini` configuration files rather than direct binary edits, giving you full control without opaque hardcoded changes.

<Note>
  As of v0.5.1, Shaiya Core is in beta. It is suitable for development and testing but not yet recommended for public production servers. Consider remaining on Shaiya Essentials until a stable release is available.
</Note>
